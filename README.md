# Fórum SB-HC
Seja bem-vindo(a) ao fórum Suíça Baiana Hacker Clube. Como você já deve ter percebido isso aqui não é um fórum tradicional. Estamos usando esse repositório para centralizar as discussões que as vezes ficam espalhadas em grupos de Whatsapp, Telegram e Facebook e que acabam se perdendo.

Algumas comunidades como Front-end Brasil (https://github.com/frontendbr/forum) e Back-End Brasil (https://github.com/backend-br/forum)  já utilizam esse modelo e tem se mostrado eficiente.

Cada issue é equivalente a um tópico e pode ser respondido normalmente como numa discussão.
